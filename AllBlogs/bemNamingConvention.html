CSS Code:

Detail Explanation:

  • First, You need to copy the above HTML code and save it in .html file extension. Example "card.html" .
  • Then you need to copy the above CSS Code and save it in .css file extension. Example : "style.css".
  • Then you need to add the following code in the html file inside the head element.
  • <link rel="stylesheet" href="/style.css" >
  • Note: Make sure that both files card.html and style.css are under same folder or same path.
  • we write block just by name . Example "class=card" is considered as block .
  • We write element by blockname__elementName. Example : class= "card__button" class="card__content" class="card__img" class="card__description" class="card__title".
  • We write modifier either by blockname--modifiername or by elementname--modifiername Example : class= "card__title--active".
